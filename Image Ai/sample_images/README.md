# Sample Images

This folder contains sample images for testing the Image AI Platform.

## Available Samples

Add your sample images here with the following naming convention:
- `sample_1.jpg` - Text document sample
- `sample_2.jpg` - Menu or signboard sample
- `sample_3.jpg` - Mixed language text
- `sample_4.jpg` - Handwritten text
- `sample_5.jpg` - Any scene for captioning

## Usage

Users can select these sample images from the web interface instead of uploading their own files.

## Adding New Samples

1. Add image files to this folder
2. Supported formats: JPG, JPEG, PNG, BMP, WEBP
3. Recommended size: 800x600 to 1920x1080
4. File names should start with "sample_"
